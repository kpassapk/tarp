Running MasterAlloc in Komodo

As with all Tarp scripts, tamasteralloc is run in Komodo as an external command; double clicking one of the icons in the MasterAlloc folder in the Tarp Toolkit project will run the script with certain command-line options. Only a few of the most common options are available through Komodo; use the program directly from the command line for other combinations.

In order to use this script from Komodo (with the exception of README, --version and --help), a .tex must be open and visible in the Komodo workspace, and there must be a Tarp Style (TAS) file called TASfile, TASfile.txt or TASfile.tas in the same directory as the .tex file.

Options

[current .tex]

Equivalent to running the following in a command prompt:

    tamasteralloc file.tex

where file.tex is the name of the currently open file. The script is run in the directory where the current file resides. If successful, a list of exercises will be shown in the "Command Output" pane in Komodo.

README,

--version

Equivalent to running the following in a command prompt:

    tamasteralloc --version
--help

Equivalent to running the following in a command prompt:

    tamasteralloc --help

TAS Entries

  • beginTag, endTag, exTag, partTag, subPartTag, sequenceRestart

    The usual LaTeXtract tags

  • masterRef

    Used for extracting the master number from each exercise.

Output

Aside from the same error output as LaTeXtract, this program produces additional error messages if master numbers are not where they are expected to be. More details about the algorithms, errors and warnings in this program can be found in the Tarp::MasterAlloc documentation.

The "Line" column contains the line numbers at the start of each exercise's range; the "Content" column contains the exercise and master number. Double clicking on a row will take open the right file in the Workspace and highlight the exercise line.
